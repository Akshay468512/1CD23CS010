# Campus Evaluation - Notification System Design

This repository contains the completed stages of the Notification System Design assessment.

## Project Structure
- **/backend**: Express-based server implementation.
- **/frontend**: React-based frontend application.
- **/notification-system-design.md**: Technical documentation covering system design, database choices, scaling strategies, and optimization queries.
- **/priority_inbox.js**: Implementation of the Priority Inbox algorithm.
- **/priority_output.png**: Screenshot of the priority inbox functionality.

## Documentation
The `notification-system-design.md` file contains the detailed response to all 6 stages of the assessment, including:
- REST API Design & WebSocket integration.
- Database Schema & Scaling strategies.
- Query optimization & performance analysis.
- Mass notification architecture.
- Priority Inbox algorithm.

## How to Run
1. Navigate to the `/backend` folder: `npm install` then `node index.js`.
2. Navigate to the `/frontend` folder: `npm install` then `npm run dev`.
